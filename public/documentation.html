How to Use the Application

  1. Open the website and select “Click Here to Start” to open the application page and begin using it.
  2. The navigation bar is blue (orange on mobile) and located directly under the title on every page and can be used to move between pages as follows:
    • Home leads to the introductory animation page
    • App leads to main application page
    • About leads to a page explaining our goals and what the app is used for
    • Documentation leads to a page describing how to use the application
  3. On the Application page, there is a search bar and a map which are the main points of interactivity.
  4. To begin a search:
    1. Click the search bar
    2. A dropdown list of categories will appear
    3. Scroll through the list or type in the text box above it to find a desired category and hit the checkbox on the left of the name to select it
    4. If the category is not in the list, it will display “Not Found”
    5. Multiple categories may be selected at one time
    6. Selected categories will be highlighted orange and the top bar will display how many have been chosen
    7. When finished, click “Apply” to see a list of results
  5. On the About and Documentation pages, you can click on any orange colored option to open up an “accordion” that contains the actual content

How to Use the Application on a Personal Server

These are steps to run the app on your personal machine with a personal server (not connecting to Heroku).

  1. Download the INST377_Team17 repository into whatever folder you want
  2. In Terminal (on Mac) or Command Prompt (on Windows), navigate to the folder where you downloaded the repository
    1. You should access the code files in this folder
  3. Enter the command: npm install
    1. Once that has finished downloading, enter the command: npm start
  4. This command will start a server on your personal machine on port 3000
  5. Open a browser and navigate to localhost:3000
  6. The app should open there
  7. This page will update in real-time every time you make changes to the code, save it, then refresh the page
  8. This is ideal for testing code and seeing the results update in real time
